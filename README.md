# 🎬 Loopin - Web

Modern ve kullanıcı dostu bir video paylaşım platformu.  
Loopin, kullanıcıların video yükleyebildiği, izleyebildiği, beğenebildiği ve etkileşimde bulunabildiği full-stack bir projedir.

---

## 🚀 Özellikler

### 👤 Kullanıcı İşlemleri
- ✅ Kayıt olma & giriş yapma
- 📧 E-posta doğrulama sistemi
- 🔐 Giriş doğrulama (token / mail onayı)
- 🔑 Şifre sıfırlama (kod ile)
- 🔒 Şifre değiştirme

---

### 🎬 Video Özellikleri
- 📤 Video yükleme
- ▶️ Video izleme
- 👍 Beğenme / beğeni kaldırma
- 👁 İzlenme sayısı takibi
- 💬 Yorum yapma & silme

---

### ⭐ Ekstra Özellikler
- 📌 Daha Sonra İzle listesi
- ❤️ Beğenilenler listesi
- 🔍 Gelişmiş arama (video + kullanıcı)
- 👤 Kanal sayfası (kullanıcı videoları)
- 🔔 Abonelik sistemi

---

### 🛠 Admin Panel
- 👥 Kullanıcı yönetimi (silme)
- 🎬 Video yönetimi (silme)
- 📊 Sistem kontrolü

---

## 🖥️ Kullanılan Teknolojiler

### 🔧 Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQLite

---

### 🎨 Frontend (Web)
- ASP.NET MVC (Razor Views)
- Bootstrap + Custom CSS

---

### ⚙️ Diğer
- SMTP Email Service
- Token tabanlı doğrulama sistemi
